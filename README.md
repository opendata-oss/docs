# OpenData Documentation

Documentation site for [OpenData](https://github.com/opendata-oss/opendata), a family of open-source databases built on a shared, object-native storage foundation powered by [SlateDB](https://github.com/slatedb/slatedb).

## Local development

Install the [Mintlify CLI](https://www.npmjs.com/package/mint) and start the dev server:

```bash
npm i -g mint
mint dev
```

Preview at `http://localhost:3000`.
